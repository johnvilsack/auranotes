Quick Setup Guide

  1. Install from Chrome Web Store: Click the "Add to Chrome" button.
  2. Connect Google Drive: Open the AuraNotes popup, click "Connect to Google Drive", and sign in.
  3. Choose Storage: Select whether to store your notes file in a hidden app folder (recommended) or a visible "AuraNotes" folder in your Drive.
  4. Start Noting: Navigate to any webpage, click the AuraNotes icon in your toolbar, and select "Add New Note". It's that simple!

For detailed Google Cloud Project setup (if you're self-hosting or modifying the source), please refer to the full setup instructions on GitHub.
